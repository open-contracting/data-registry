Word choice
-----------

This project's names are inconsistent with other projects:

-  "collection”" has a different meaning in the code than in Kingfisher Collect or Kingfisher Process. It should be “publication”, as used in the UI and documentation.
-  Foreign key `related_name <https://docs.djangoproject.com/en/5.0/ref/models/fields/#django.db.models.ForeignKey.related_name>`__ values are singular, instead of plural.
